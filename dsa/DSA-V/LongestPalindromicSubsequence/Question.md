Problem Description

Given a string s, find the length of the longest palindromic subsequence in s.

A subsequence is a sequence that can be derived from another sequence by deleting some or no elements without changing the order of the remaining elements.
Problem Constraints

    1 ≤ |s| ≤ 1000
    s consists of lowercase English letters ('a'–'z').
    Output must be a single integer with no extra spaces.

Input Format

    A single line containing the string s.

Output Format

    A single integer: the length of the longest palindromic subsequence in s.

Example Input 1

bbbab

Example Output 1

4

Example Explanation 1

The longest palindromic subsequence is "bbbb", which has length 4.
Example Input 2

cbbd

Example Output 2

2

Example Explanation 2

The longest palindromic subsequence is "bb", which has length 2.
